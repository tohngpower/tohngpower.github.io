Cost and safety

Select Google AI now free to use the shared Gemini 2.0 API key but note that usage is limited and your data may be used for AI training. Alternatively, you can obtain your own Google AI API key and pay as you go, ensuring your data is not used to train AI. Additional AI model: LLama 3.2, 3.3 and Typhoon which are free to use with cetain rate limit to improve availability.

Frequently Asked Questions

A useful FAQ section.

How do I get API key for free?

Now Gemini 2.0 flash and Gemini 2.0 flash thinking exp are free to use with some limitation but you have to create project in Google cloud to create API key.

AI cannot speak Japanese.

You can click "settings" button and change langauge to Japanese. *Noted: App can support other languages > 20 for both text and speech.

How to create an image?

You can type "imageGEN" and follow with your prompt for image details.

How do we know how many tokens using now?

You can click "check usage" button.
